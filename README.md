( JustHike ) JustHike backend is a TypeScript-powered Node.js/Express API serving a hiking discovery platform. Features include JWT authentication, role-based access (hikers, guides, admins), trek management, booking system, and blog CRUD. Implements clean architecture with repository pattern for maintainability. Secure password reset via Nodemailer, 80+ Jest tests ensuring reliability, and PostgreSQL/MongoDB for data persistence. Built for scalability with clear separation of concerns.
